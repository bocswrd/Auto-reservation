### 概要
1. Pythonをインストールする
1. Chromeをインストールする
1. [仮想環境を作成する](#仮想環境の作成方法)
1. [パッケージをインストールする](#パッケージのインストール方法)

### 仮想環境の作成方法
``` bash
python -m venv .venv
```

### パッケージのインストール方法
``` bash
pip install -r requirements.txt
```
